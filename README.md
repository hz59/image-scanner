Image-Scanner 👁‍🗨
===
Open-source project that use Drype to create a Vulnerability scan report for a choosen image.
## Informations  ☕️
- Title:  `image-scanner`
- Author:  `hz59`
- email: [coming soon]()

## Install & Dependence  🚀
- python3
- grype: https://github.com/anchore/grype.git 

## Getting started ▶️
- Run:
  ```
  python3 txt2html.py
  ```
- Enter the name of the image you need to scan and click enter:
  ```
  Please enter the name of the image to scan:
  ```
- Wait until the scan is completed and open this file in a browser:
  ```
  output.html
  ```
## Directory Hierarchy
```
|—— output.html
|—— scan-result.txt
|—— txt2html.py
```
